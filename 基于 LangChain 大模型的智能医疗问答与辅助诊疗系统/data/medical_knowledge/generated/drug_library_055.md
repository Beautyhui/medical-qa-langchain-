# 药品参考知识库

### 药品1: 湿疹散


**类型**: 西药

**相关疾病**: 接触性皮炎 && 湿疹 && 药物性皮炎 && 药疹 && 婴儿湿疹

**适应症**: 用于急慢性湿疹、婴儿湿疹、药物性皮炎(药疹)、接触性皮炎、脓疱疮等；对下肢溃疡等也有一定疗效。

**用法用量**: 外用：按部位大小，取药适量，茶水或麻油调敷患处，或用水煎湿敷。

### 药品2: 盐酸奥洛他定片


**类型**: 西药

**适应症**: 过敏性鼻炎、荨麻疹、瘙痒性皮肤病（湿疹、皮炎、痒疹、皮肤瘙痒症、寻常性银屑病、渗出性多形红斑）

**用法用量**: 成人用量通常为1日2次，1次5mg，早晨和晚上睡前各服1次。根据年龄及症状适当增减。

**禁忌**: 对本品成份有过敏史的患者禁用。

**不良反应**: 在国外注册试验及上市后调查（包括长期使用调查）的9620例患者中，有1056例（发生率11.0%）发生不良反应或实验室检查值异常，共计1402例次。主要不良反应为嗜睡674例次（7.0%），丙氨酸氨基转移酶（ALT）升高68例次（0.7%），倦怠感53例次（0.6%），天门冬氨酸氨基转移酶（ASL）升高46例次（0.5%），口渴36例次（0.4%）等。1）严重不良反应肝功能损害、黄疸（发生率不明）：有可能发生天门冬氨酸氨基转移酶（AST）、丙氨酸氨基转移酶（ALT）、γ-谷氨酰转肽酶（γ-GTP）、乳酸脱氢酶（LDH）、碱性磷酸酶（ALP）升高和黄疸，故应注意观察。若出现异常，应停药并进行适当处置。2）其他不良反应有可能发生以下不良反应，故应注意观察，发现异常时应进行减量或停药等适当处置。不良反应症状和发生率 注）出现此类症状时应停止服药。

**药物相互作用**: 尚不明确。

### 药品3: 碘化钾


**相关疾病**: 单纯性甲状腺肿 && 地方性甲状腺肿 && 甲亢

**适应症**: 1.预防和治疗地方性甲状腺肿。2.治疗单纯性甲状腺肿。3.治疗甲亢危象。4.甲亢术前准备用药。

**用法用量**: 1.治疗单纯性甲状腺肿：从小剂量开始：每天10mg，20天为1个疗程，2个疗程后休息30～40天，剂量增至20～25mg，总疗程3～6个月。2.预防地方性甲状腺肿：每天100μg，多采用食盐加碘的方法治疗地方性甲状腺肿，每天1～10mg，连用1～3个月，休息30～40天，剂量逐渐增至20～25mg，总疗程3～6个月；也可服用复方碘溶液1～2滴，每天1次，连服1个月，休息10天后再服。3.甲亢术前准备：于术前2周服复方碘溶液，每天3次，每次从5滴开始，逐渐增加至15滴。4.治疗甲亢危象：首剂用复方碘溶液60滴（3.6ml），以后每6小时口服30～45滴。

**禁忌**: 1.过敏者禁用。2.孕妇避免摄入过量碘，哺乳期亦不应服用。

**不良反应**: 过敏反应，有少数人出现血管神经性水肿，上呼吸道黏膜及喉头水肿，严重时引起窒息。

**药物相互作用**: 增加盐的摄入或大量饮水，均能增加碘的排出。

**说明**: 长期小剂量补充碘可防止地方甲状腺肿的发生，大剂量，短期应用碘剂具有封闭甲状腺的作用，可防止已经合成的甲状腺素的继续释放，缓解甲状腺危象病情，并且能防止手术中甲状腺的释放，防止术后甲状腺危象的发生。碘剂是甲亢术前准备不可缺少的药物，对碘过敏者很少见，在应用过程中注意防止过量而诱发甲亢。

### 药品4: 磷脂苯芥


**相关疾病**: 鼻咽癌 && 肺癌 && 乳腺癌 && 上颌窦癌 && 胸腔积液 && 原发性肝癌

**适应症**: 1.鼻咽癌、上颌窦癌及口腔肿瘤。2.脑瘤、肺癌、乳腺癌、原发性肝癌及淋巴瘤。3.癌性胸腔积液。

**用法用量**: 1.口服：成人每天20～30mg，分3次服，10～14天为1个疗程。2.静脉滴注：5～10mg，每天或3天1次，10～14次为1个疗程。3.腔内注射：每次20～30mg，每周1～2次(可抽出积液后，再腔内注射)。

**禁忌**: 与一般烷化剂相同。

**不良反应**: 1.胃肠道反应有恶心、呕吐及食欲减退等。用药前给予止吐剂、镇静剂及碳酸氢钠，可减轻上述不良反应。2.骨髓抑制较氮芥小，白细胞下降明显，停药后可恢复，对血小板及血红蛋白影响较轻。3.偶见脱发，停药后可恢复。

**药物相互作用**: 与一般烷化剂相同。

**说明**: 对鼻咽癌、上颌窦癌及口腔肿瘤疗效较好。对脑瘤、肺癌、乳腺癌、肝癌及淋巴瘤有一定疗效。对癌性胸腔积液疗效显著。

### 药品5: 阿昔莫司


**类型**: 西药

**相关疾病**: 高脂蛋白血症 && 高脂蛋白血症Ⅲ型 && 高脂蛋白血症Ⅳ型 && 高脂蛋白血症Ⅴ型

**适应症**: 高脂蛋白血症Ⅱa型、高脂蛋白血症Ⅲb型、高脂蛋白血症Ⅲ型、高脂蛋白血症Ⅳ型、高脂蛋白血症Ⅴ型。

**用法用量**: 饭后服药。对Ⅳ型患者每次250mg，每天3次；对Ⅱ、Ⅲ、Ⅳ型患者可每次250mg，每天3次，必要时最大量可用到每天1 200mg。

**禁忌**: 对本品过敏者、孕妇及哺乳期妇女禁用，严重溃疡病、肝功能严重损害者禁用。

**不良反应**: 不良反应明显小于烟酸。目前尚未发现有明显的肝、肾功能损害情况；未见有明显的代谢紊乱现象；面部潮红、皮肤瘙痒的发生率约6%。个别患者有上腹不适、胃灼热、恶心、腹泻。这些不良反应多在服药几日后逐渐自行减轻或消失。

**药物相互作用**: 与口服降糖药、口服抗凝药合用尚未见有药物相互作用。

**说明**: 临床研究表明，阿昔莫司是一种安全、有效的血脂调节药物，可使TC降低25%，TG降低50%，HDL-C升高20%。

### 药品6: 亚锡焦磷酸钠


**适应症**: 本品可用于制备锝[99mTc]焦磷酸盐注射液，但目前临床上大多用于锝[99mTc]标记红细胞。

**用法用量**: 用于制备锝[99mTc]焦磷酸盐注射液，取本品1瓶(支)，在无菌操作条件下，将高锝[99mTc]酸钠注射液4～6ml注入瓶中，充分振摇，使冻干物溶解，室温静置5分钟即制得[99mTc]-PYP注射液。用于标记红细胞，取本品1瓶，在无菌操作条件下，将生理盐水2～5ml注入瓶中，充分振摇，使冻干物溶解，给预先口服了400mg过氯酸钾的受检者静脉注射。20～30分钟后，再静脉注射高锝[99mTc]酸钠注射液370～740MBq(10～20mCi)。20～30分钟后，即完成锝[99mTc]标记红细胞(体内法)。

**禁忌**: 孕妇及哺乳期妇女禁用。

**不良反应**: 偶有皮疹、瘙痒、荨麻疹等过敏反应。

### 药品7: 人源化地特胰岛素


**相关疾病**: 糖尿病

**适应症**: 1．用于治疗成人及儿童1型糖尿病。2．也可用于2型糖尿病患者补充基础长效胰岛素以控制其高血糖。

**用法用量**: 皮下注射，每日1～2次，剂量根据血糖浓度而定。口服降糖药不能控制的2型糖尿病，如果患者未使用过胰岛素治疗，本品的起始剂量为0.1～0.2U/kg，每日1次，或每次10U，每日1～2次。正在接受普通胰岛素治疗的1型和2型糖尿病患者改用本品时，可进行剂量换算。

**禁忌**: 1．对本品过敏者禁用。2．生殖毒性分级为C，不推荐孕妇使用。

**不良反应**: 1．全身：过敏反应。2．皮肤及附属：瘙痒、药疹、脂肪营养障碍。3．注射部位反应：本品的注射部位反应发生率比普通胰岛素高，可出现局部发红、疼痛、瘙痒、荨麻疹、肿胀、烧灼感等。4．其他：低血糖、体重增加、水钠潴留和水肿。5．本品剂量不足可引起血糖升高，导致1型糖尿病患者糖尿病酮症酸中毒，出现恶心、呕吐、困倦、皮肤发红、口干、排尿增加、口渴、食欲不振、丙酮味口臭等。

**药物相互作用**: 1．与以下药物合用，本品的降糖作用降低：皮质激素、达那唑、利尿剂、拟交感神经药物、异烟肼、吩噻嗪类药物、生长激素、甲状腺激素、雌激素、孕激素等。2．与以下药物合用，本品的降糖作用增强：口服降糖药、血管紧张素转化酶抑制剂、丙吡胺、氯贝特、氟西汀、单胺氧化酶抑制剂、右丙氧芬、水杨酸、生长激素及磺胺类抗菌药物。3．β受体阻滞剂、可乐定、锂盐和酒精增强或降低本品的降血糖作用。4．与喷他脒合用可引起低血糖，有时伴随高血糖。5．与抗交感活性药物(如β受体阻滞剂、可乐定、胍乙啶、利血平)合用，低血糖症状可能减轻或被掩盖。6．本品可使速效胰岛素的AUC和Cmax降低约40%，故本品不与其他胰岛素制剂混合使用。

### 药品8: 头孢羟氨苄


**相关疾病**: 急性扁桃体炎 && 尿路感染

**适应症**: 适用于治疗敏感细菌所致的尿路感染、皮肤软组织感染、急性扁桃体炎、急性咽炎、中耳炎及肺部感染等。

**用法用量**: 口服给药：1.成人每次0.5～1g，每天2次。最大剂量为每天4g。2.肾功能不全时剂量：肾功能减退者首次剂量为1g饱和量，然后根据肾功能减退程度延长给药间期。肌酐清除率为每分钟25～50ml、10～25ml和0～10ml时，分别每12小时、24小时和36小时服药500mg。3.儿童：(1)一般感染：每12小时15～20mg/kg。(2)A组溶血性链球菌咽炎及扁桃体炎：每12小时15mg/kg，共10天。

**禁忌**: 对本药或其他头孢菌素类药过敏者禁用。

**不良反应**: 头孢羟氨苄的不良反应少而轻微，且一般呈暂时性和可逆性。不良反应总发生率约为4%，以胃肠反应为主。1.过敏反应：以皮疹、荨麻疹、红斑、药物热等过敏反应症状较多见，过敏性休克少见。2.消化道反应：以恶心、呕吐、腹泻和腹部不适等胃肠道症状较为多见，偶见假膜性肠炎。3.肝毒性：少数患者用药后可出现血清丙氨酸氨基转移酶、天门冬氨酸氨基转移酶和碱性磷酸酶一过性升高。4.肾毒性：少数患者用药后偶可出现尿素氮、肌酸、肌酐暂时性升高。5.泌尿生殖系统：曾有用药过程中出现阴道瘙痒、烧灼感、排尿困难或尿频的报道。

**药物相互作用**: 1.与丙磺舒同用可延迟本药的排泄，升高本药的血药浓度。2.与呋塞米、布美他尼、依他尼酸等利尿药合用可增加肾毒性。3.与氨基糖苷类抗生素同用可增加肾毒性。4.与伤寒活疫苗同用可能降低伤寒活疫苗的免疫效应，其可能的机制是本药对伤寒沙门菌具有抗菌活性。

**说明**: 本品的抗菌活性与头孢氨苄相似。本品给药方便，不良反应少而轻。主要治疗呼吸系统、泌尿系统、皮肤与软组织、骨与关节的感染及中耳炎，疗效满意。

### 药品9: 戈洛帕米


**类型**: 西药

**相关疾病**: 充血性心力衰竭 && 低血压 && 慢性心功能不全 && 心律失常

**适应症**: 防治心绞痛，充血性心力衰竭(慢性心功能不全)，治疗心肌梗死后的无节律性心动过速和一般心律失常。

**用法用量**: 口服25～50mg，6～12小时一次，日极量为200mg。

**禁忌**: 孕妇、哺乳者、心功能不全、严重肝肾功能不全、低血压及传导阻滞。

**不良反应**: 胃肠道不适、便秘、心动过缓及传导阻滞。

### 药品10: 替莫泊芬


**相关疾病**: 结肠癌 && 直肠癌

**适应症**: 本品对胸膜瘤、弥漫性间皮瘤、口腔肿瘤、结肠癌及直肠癌均有良好疗效，并可有效预防恶性肿瘤的复发。

**不良反应**: 本品与第一代光敏剂相比副作用小、皮肤光敏性低，在剂量降低的同时，毒性也大大降低。m-THPC介导的光动力学治疗并不会使肿瘤细胞对随后的光动力治疗(photodynamic therapy，PDT)、化疗和放疗产生耐受性。另外，不经过光照的m-THPC几乎没有细胞毒性作用。

### 药品11: 槐耳颗粒


**类型**: 中药

**相关疾病**: 腹胀 && 原发性肝癌

**适应症**: 扶正、活血、抑瘤。用于不宜手术和化疗的原发性肝癌的辅助治疗，可改善肝区疼痛、腹胀、乏力等症状。

**用法用量**: 口服，一次20g，一日3次。一个月为1个疗程，或遵医嘱。肺癌、胃肠癌和乳腺癌的辅助治疗六周为一个疗程。

**禁忌**: 尚不明确。

**不良反应**: 个别患者出现恶心，呕吐。

### 药品12: 氟地西泮


**类型**: 西药

**相关疾病**: 麻痹 && 失眠 && 重症肌无力

**适应症**: 用于身心疾病、自主神经功能紊乱、脑性麻痹等疾病所致的焦虑、紧张、抑郁及失眠等焦虑状态和焦虑症。

**用法用量**: 口服：成人每天0.75mg，分3次服，并可根据病情和年龄增减。

**禁忌**: 急性闭角型青光眼及重症肌无力患者禁用。

**不良反应**: 偶有口渴、食欲缺乏、疲劳及皮疹等不良反应。罕见黄疸、性欲减退、排尿困难及毛发脱落等。长期大量口服可产生依赖性。急剧减量或停药可出现戒断症状。

**药物相互作用**: 与吩噻嗪类、巴比妥类及单胺氧化酶抑制药合用，可使其作用加强。

**说明**: 本品为抗焦虑药。能消除患者焦虑、紧张作用强、很小剂量即有效、其抗焦虑作用比西泮强、但镇静、催眠作用比地西泮小。

### 药品13: 焕雌醇/去氧孕烯


**类型**: 西药

**相关疾病**: 乳腺癌 && 血栓形成

**适应症**: 临床作为避孕药效果可靠、周期控制好，升高HDL，有利于脂质代谢，不增加体重，无雄性征等副作用。

**用法用量**: 口服复方去氧孕烯片，从月经第1天开始，1片/d，不间断连服21天，然后停药7天；第29天起开始服下1个周期药片。

**禁忌**: 严重肝功能障碍、血栓形成或血栓、乳腺癌、子宫癌及哺乳期忌用。

**不良反应**: 可有不规则出血，开始发生率较高，半年后逐渐减少；尚可见恶心、头痛、闭经、乳胀、乏力、抑郁等。

**药物相互作用**: 药物相互作用：利福平、巴比妥类、苯妥英钠等可使本品和炔雌醇活性降低。

**说明**: 去氧孕烯临床作为避孕药效果可靠、周期控制好，升高HDL，有利于脂质代谢，不增加体重，无雄性征等副作用。要求从月经第1天开始，不间断连服21天，然后停药7天。严重肝功能障碍、血栓形成或血栓、乳腺癌、子宫癌及哺乳期忌用。不良反应半年后逐渐减少。

### 药品14: 多西他赛


**相关疾病**: 肺癌 && 乳腺癌 && 胃癌

**适应症**: 用于治疗晚期乳腺癌、非小细胞肺癌。也有报道用于治疗头颈部癌、小细胞肺癌、胃癌、卵巢癌等肿瘤。

### 药品60: 盐酸丁卡因胶浆


**类型**: 西药

**适应症**: 本品为腔道表面润滑麻醉剂，用作尿道、食道、阴道、肛门、直肠等插管镜检或手术时的局部润滑麻醉。

**用法用量**: 外用。一次2～5g，插管、镜检或手术前用。1、用于做胃镜检查或食管扩张时，可将本品2g左右滴于患者舌根部，令患者做吞咽动作，立即起麻醉作用；同时将本品适量涂于胃镜管或扩张器的表面(起润滑作用)即可操作。2、用于做喉或声带检查时可按上述方法进行。3、用于男性尿道时(如尿道扩张、膀胱镜检查、逆行肾盂造影、经尿道进行前列腺切除术、导尿术等)，先将尿道口洗净消毒，将软管插入尿道，将本品挤入(约5g)，以阴茎夹夹住，2分钟后即可插入膀胱镜等器械，进行镜检或手术。4、用于女性尿道时基本方法同3。5、用于妇科阴道检查时，将软管插入阴道，挤入本品约3g，同时可在扩张器或其他器械上涂上本品少许以增加润滑减少阴道损伤。6、用于人工流产时，可将本品挤在宫颈口上，2分钟左右宫颈松弛，即可手术。7、用于直肠镜检可将软管插入肛门，挤出本品约3g，同时在扩张器或其他器械上涂上本品少许，即可进行检查。肛门直肠镜检需扩肛时本品不能达到肛门括约肌的松弛的目的。8、用于肛门或肛裂时，直接将本品涂在肛门上即可。

**禁忌**: 1、高过敏体质患者禁用。2、腔道破裂、血管外露者禁用。

**不良反应**: 偶见过敏反应。

**药物相互作用**: 1、本品属酸性，不得与碱性药混合在一起使用；即使酸性要，由于PH值不同，也可影响其解离值，以致麻醉减效，或起效时间迟延。2、与磺胺族药同用时，后者即减效。3、本品与普鲁卡因、肥皂、碘化钾、硼砂、碳酸、碳酸氢盐、碳酸盐、氧化物、枸橼酸盐、磷酸盐和硫酸盐配伍禁忌。
