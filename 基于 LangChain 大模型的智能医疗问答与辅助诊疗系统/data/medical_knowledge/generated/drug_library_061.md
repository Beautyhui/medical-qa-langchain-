# 药品参考知识库

### 药品1: 溴莫尼定


**类型**: 西药

**相关疾病**: 高眼压 && 高眼压症

**适应症**: 开角型青光眼和高眼压症，预防氩激光小梁成形术后的高眼压以及无法应用β受体阻断药患者。

**用法用量**: 点眼：每天2次，如眼压在下午达高峰的患者或眼压须另加控制的患者，下午可增加1滴。

**禁忌**: 1.对本品过敏者禁用，儿童禁用。2.使用单胺氧化酶抑制药患者禁用。

**不良反应**: 1.口干、烧灼感、刺痛感、眼部充血、头痛、视物不清、眼异物感、疲劳或倦怠、结膜疱疹、眼瘙痒。2.少见眼部干燥、流泪、畏光、疼痛、头晕，结膜水肿、变白以及角膜染色等。3.偶见失眠、结膜分泌物增多或出血、精神抑郁、高血压、焦虑、心悸及晕厥等。

**药物相互作用**: 对中枢神经系统抑制药(巴比妥类、鸦片制剂、镇静剂或麻醉剂)和酒精产生叠加作用或使之强化的可能性，须予以考虑。

**说明**: 本品为α2受体激动药。具相对选择性，用药后2h降眼压达最大效应。具有双重作用机制；可减少房水的生成(类似于β受体阻断药)，又增加房水经虹膜、巩膜的流出(类似于前列腺素)。降眼压效果与噻吗洛尔相似。主要用于开角型青光眼和高眼压症。

### 药品4: 盐酸氨基葡萄糖片


**类型**: 西药

**适应症**: 全身所有部位骨关节炎的治疗和预防：包括膝关节、髋关节、脊柱、肩、手和手腕、踝关节等。

**用法用量**: 口服。一次240mg～480mg(1～2片)，一日3次，或遵医嘱。根据病人病情，连续服用4～12周，如有必要在医生指导下可延长服药时间。每年重复治疗2～3次。

**禁忌**: 对氨基葡萄糖过敏的病人。

**不良反应**: 罕见有轻度的胃肠道不适，如恶心、便秘、腹胀和腹泻。有些病人可能出现过敏反应，包括皮疹、瘙痒和皮肤红斑。

**药物相互作用**: 目前尚缺乏详细的研究资料。

### 药品7: 乳酶生


**类型**: 西药

**相关疾病**: 腹泻 && 腹胀

**适应症**: 用于消化不良，肠道菌群失调或肠内异常发酵引起的腹胀、腹泻，小儿饮食不当引起的腹泻等。

**用法用量**: 口服给药 每次0.3～0.9g，每天3次。

**禁忌**: 对本药过敏者禁用。

**不良反应**: 本药无明显不良反应。

**药物相互作用**: 1.氨基酸、干酵母与本药联用，可增强药效。2.抗菌药物(如红霉素、氯霉素、土霉素等)或吸附剂(如药用炭)可使本药失活，降低本药的疗效，故两者不宜合用，如必须服用则应间隔2～3h。3.鞣酸蛋白、铋剂、酊剂、喹碘方、氯化亚汞可抑制、吸附或杀灭乳酸杆菌，降低本药疗效。

**说明**: 本品为干燥活乳酸菌制剂。主要用于消化不良、小儿饮食不当引起腹泻可为首选药。本品宜冷暗处贮存，以免减低效力。且须在有效期内使用。

### 药品10: 副醛


**类型**: 西药

**相关疾病**: 癫痫持续状态 && 破伤风 && 失眠

**适应症**: 1.可用于失眠。2.尚有抗惊厥作用，用于破伤风、子痫、癫痫持续状态及解救惊厥药中毒。

**用法用量**: 1.口服给药：每次5～10ml(可放入牛奶或果汁中饮用)。2.静脉注射：每次1～2ml(只供急诊时使用，用生理盐水稀释)。3.直肠给药：每次5～10ml(用温开水稀释至30～50ml)。4.肌内注射：每次2～5ml(臂部深处注射)。

**禁忌**: 1.产科麻醉。2.支气管肺病。3.肝功能不全。4.胃肠炎，尤其有溃疡者。5.对本药高度过敏者。

**不良反应**: 口服对胃肠黏膜有刺激性。肌内注射时偶可引起无菌性脓肿和永久性坐骨神经损伤。静脉用药，已出现肺出血、肺水肿、循环衰竭、呼吸窘迫和其他严重后果。长期使用可产生依赖性。剂量过大可抑制呼吸。本药已有引起死亡的报道。

**药物相互作用**: 1.在动物模型中，副醛与双硫仑合用可引起双硫仑-酒精样反应。两者禁止合用。2.月见草油可降低癫痫发作阈，增加惊厥的发生率，故月见草油不宜与本药合用。3.某些银杏制剂中含有具有神经毒性的4′-O-甲基吡哆醇，如果其数量足够多，就可导致惊厥，从而拮抗副醛的作用。4.副醛与酒精合用，易导致代谢性酸中毒。故接受副醛治疗的患者应避免饮酒。5.对于癫痫持续状态用其他药物治疗无效时可经直肠给药(国外注射用剂型已停止生产)。直肠给药常用于儿童，但这种方式给药时剂量难以控制，且吸收很慢。6.与副醛不相容的药物有：盐酸氯丙嗪、丙氯拉嗪。7.副醛溶液可使聚乙烯静脉内插管变软或将其溶解。8.应避免与橡胶或塑料制品接触，只能用玻璃注射器和玻璃容器。9.副醛暴露于光或空气中可自然解聚为乙醛。为避免分解，所有形式的副醛都应贮藏于阴凉避光处，最好是冷藏。微黄色的副醛溶液已发生降解，若出现乙酸气味，也表明发生了氧化分解，均应弃去不用。

**说明**: 作用同水合氯醛相似，有镇静、安眠与抗惊厥作用，但作用比水合氯醛强。作用可维持8h。

### 药品11: 双贝特


**类型**: 西药

**相关疾病**: 高脂血症

**适应症**: 用于治疗高脂血症，尤适用Ⅳ型高脂血症的治疗。心脉康糖衣片可用于心绞痛、高血压的治疗。

**用法用量**: 口服：每次0.25～0.5g，每日3次，6周为1个疗程。

**禁忌**: 肝肾功能不全者及孕妇忌用。

**不良反应**: 本品毒性较低，偶可引起恶心、呕吐、腹泻等胃肠道反应，但比氯贝丁酯轻。个别患者可有谷丙转氨酶暂时升高，不必停药，可自行恢复。

### 药品14: 尼莫司汀


**相关疾病**: 恶性淋巴瘤 && 肺癌 && 胃癌 && 原发性肝癌

**适应症**: 用于治疗脑瘤、肺癌、胃癌、肠癌、原发性肝癌和恶性淋巴瘤等，可与其他抗癌药物合并使用。

**用法用量**: 1.每次剂量为2～3mg/kg，以注射用蒸馏水溶解(5mg/ml)，静脉注射；或者加生理盐水或5%葡萄糖液250ml，静脉滴注，6周后重复使用，总剂量300～500mg；或每次2mg/kg，每周1次，连用3次。2.本品还可用于胸腹腔注射、动脉注射和膀胱腔内给药。

**禁忌**: 有骨髓抑制者及对本药有严重过敏史者禁用。

**不良反应**: 1.有食欲缺乏、恶心、呕吐、乏力、发热、脱发和皮疹。2.对肝功能有一定影响，用药后1～3周转氨酶可升高，2～3周后自然恢复。3.并有迟发性骨髓抑制，包括白细胞减少和血小板减少，分别于4～6周后和3～6周后呈最低值。治疗中应根据血象变化来调整用药剂量。

**药物相互作用**: 与抑制骨髓作用的抗癌药联合化疗或与放疗并用时有骨髓重叠毒性。

**说明**: ACNU为水溶性亚硝脲类药物，能通过血-脑屏障，对脑瘤或脑转移癌可有一定疗效。本品不良反应中有迟发性骨髓抑制，故停药6周后始可再次给药，并应密切观察血象变化来调整用药。

### 药品15: 氨力农注射液


**类型**: 西药

**适应症**: 适用于各种类型的充血性心力衰竭，尤其对洋地黄、利尿剂、血管扩张剂治疗无效的心衰患者。

**用法用量**: 氨力农注射液用适量生理盐水稀释后，一般先以0.5~1.0mg/kg静脉注射5~10分钟，再以5~10ug/kg.min的速度静脉滴注，必要时30分钟后再静脉注射0.5~1.0mg/kg一次，每日总剂量不宜超过5~10mg/kg，医师因根据病情随时调整剂量。

**禁忌**: 严重低血压禁用。

**不良反应**: 主要表现为胃肠反应、血色素、血小板减少、室性心律失常、低血压及肝肾功能损害。另可表现为心悸、胸闷。个别病人可出现发热、皮疹等过敏反应，偶有胸痛及注射局部刺激

**药物相互作用**: 1.本品需用生理盐水稀释后使用，不能用葡萄糖稀释。2.与速尿（呋塞米）混用立即产生沉淀。

### 药品16: 注射用硫酸培洛霉素


**类型**: 西药

**适应症**: 本品主要适用于头颈部恶性肿瘤、皮肤癌、肺癌(鳞状细胞癌)、前列腺癌、恶性淋巴肿瘤等。

**用法用量**: 剂量应该从低剂量开始，逐渐增加剂量并观察治疗反应。有昏睡出现时，须将每日剂量的大部分分配至睡前服用或减量。服药第一周内症状即有所缓解，两周内出现较佳抗抑郁效果，通常需要服药两周至四周才出现最佳疗效。建议成人初始剂量每日50～100mg(1～2片)，分次服用，然后每三至四天可增加50mg(1片)。门诊病人一般每口200mg(4片)为宜。住院病人较严重者剂量可较大。最高剂量不超过每天400mg(8片)，分次服用。长期使用的维持剂量保持在最低剂量。一旦有足够的疗效，可逐渐减量。一般建议起效后疗程应持续数月。

**禁忌**: 对盐酸曲唑酮过敏者禁用，肝功能严重受损、严重的心脏疾病或心律失常者禁用，意识障碍者禁用。

**不良反应**: 常见不良反应为嗜睡、疲乏、头昏、失眠、紧张和震颤等；以及视物模糊、口干、便秘。少见体位性低血压(进餐时同时服药可减轻)、心动过速、恶心、呕吐和腹部不适。极少数病人出现肌肉骨骼疼痛和多梦。临床研究中曾报道一些不良反应可能与盐酸曲唑酮的使用有关：静坐不能、过敏反应、贫血、胃胀气、排尿异常、性功能障碍和月经异常等。但见之于为数甚少的患者。

**药物相互作用**: 1、用盐酸曲唑酮的患者，如果同时合用地高辛或苯妥因，可使地高辛或苯妥因的血浆浓度水平升高。 2、本品可能会加强对酒精、巴比妥类药和其他中枢神经抑制剂的作用。 3、本品与MAO1之间的相互作用目前尚不清楚，若刚停药后就服用本品或与其同时服用，盐酸曲唑酮应从低剂量开始，直到临床疗效的产生。 4、盐酸曲唑酮与降压药合用，需要减少降压药的剂量。
