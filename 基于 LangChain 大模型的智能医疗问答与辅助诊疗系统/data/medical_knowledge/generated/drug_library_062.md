# 药品参考知识库

### 药品1: 西黄丸


**类型**: 中药

**适应症**: 1.用于各种恶性肿瘤、急性白血病的辅助治疗。2.用于治疗淋巴结核、化脓性骨髓炎等。

**用法用量**: 丸剂，每次3g，每天两次，温开水或黄酒送服。

**说明**: 本品用于各种恶性肿瘤的辅助治疗，有一定的增效作用。对不宜手术及放、化疗的晚期患者，使用本品配合扶正治疗有一定疗效。

### 药品2: 叶绿素铜钠乳膏


**类型**: 西药

**适应症**: 用于治疗冻疮、烧伤、烫伤急性或慢性皮肤溃疡、褥疮，可用于革兰阳性菌感染引起的溃疡。

**用法用量**: 外用。一日1～4次，涂于患处。

**不良反应**: 可有皮肤过敏反应。

**药物相互作用**: 尚不明确。

### 药品3: 复方次硝酸铋


**类型**: 西药

**相关疾病**: 十二指肠溃疡 && 胃溃疡

**适应症**: 主要用于胃溃疡、十二指肠溃疡、胃酸过多、神经性消化不良、胃炎、胃灼热及痉挛等疾病。

**用法用量**: 口服：每次2片，3次/d，饭后嚼碎服或将药片掰成小片吞服。连续服2个月，病情严重者应连续服3个月，服完一个疗程后，为防止复发可将剂量减至每天1～2片，继续服用2～3个月。

**禁忌**: 胃酸缺乏患者忌用。

### 药品4: 复方氨基酸(3AA)


**相关疾病**: 肝性脑病 && 肝硬化

**适应症**: 1.用于肝性脑病、慢性活动性肝炎、重症肝炎及肝硬化。2.也可用于肝胆外科手术前后。

**用法用量**: 成人常规剂量：静脉滴注：一日250～500ml，或用适量5%～10%葡萄糖注射液混合后缓慢滴注。

**禁忌**: 尚不明确。

**不良反应**: 静脉滴注过快时，可致心悸、恶心、呕吐、发热等。

**药物相互作用**: 尚不明确。

### 药品5: 牛黄蛇胆川贝液


**类型**: 中药

**适应症**: 用于外感咳嗽中的热痰咳嗽、燥痰咳嗽，症见咳嗽痰多，或喉中有痰鸣，质粘厚，咯吐不爽。

**用法用量**: 口服。一次10毫升，一日3次，小儿酌减或遵医嘱。

**禁忌**: 恶寒发热者忌服。

**药物相互作用**: 如与其他药物同时使用可能会发生药物相互作用，详情请咨询医师或药师。

### 药品6: 痛经丸


**类型**: 西药

**相关疾病**: 腹痛 && 慢性盆腔炎 && 子宫肌瘤 && 子宫内膜异位

**适应症**: 用于寒凝血滞，经来腹痛，可治疗慢性盆腔炎、子宫内膜异位、子宫肌瘤、慢性肝炎等病症。

**用法用量**: 口服：每天1～2次，每次6～9g，温开水送服。

### 药品7: 盐酸二氢埃托啡舌下片


**类型**: 西药

**适应症**: 本品仅限用于创伤、手术后及诊断明确的各种剧烈疼痛的止痛，包括对吗啡或哌替啶无效者。

**用法用量**: 舌下含化。一次20～40μg，经10～15分钟疼痛可获明显减轻，视需要可于3～4小时后重复用药。只可舌下含化，不可将药片吞服，否则影响止痛效果。允许使用最大剂量一般为一次60μg，一日180μg，连续用药不得超过3天。超大剂量使用时应遵医嘱。

**禁忌**: 1. 脑外伤神志不清或肺功能不全者禁用。2. 婴幼儿、未成熟新生儿禁用。3. 非剧烈疼痛病例如牙痛、头痛、风湿痛、痔疮痛或局部组织小创伤痛等不宜使用。

**不良反应**: 本品用于各种疼痛病例时，在治疗剂量下一般无明显不良反应，少数病人可出现头晕、恶心、呕吐、乏力、出汗，卧床病人比活动病人反应轻。这些反应可不经任何处理而缓解。偶见呼吸减慢至每分钟10次左右，用呼吸兴奋药尼可刹米可纠正，也可用吸氧纠正。未见吗啡样致便秘作用。本品对循环系统的功能影响很小。本品有耐受性及依赖性，连续多次使用本品后，止痛持续时间缩短。

**药物相互作用**: 未进行该项实验且无可靠参考文献。

### 药品8: 苯溴马隆


**相关疾病**: 痛风

**适应症**: 主要用于慢性痛风的治疗。也可用于各种原因引起的继发性高尿酸血症，原发性高尿酸血症。

**用法用量**: 口服，每天1次，每次50mg，逐渐增加剂量，连续3～6个月。早饭后与碳酸氢钠3g同服，以保持尿液碱化，利于尿酸结晶溶解。每天最少饮1.5～2L水。维持量每天100～150mg。

**禁忌**: 1.有中、重度肾功能不全的患者禁用。2.孕妇禁用。

**不良反应**: 本品不良反应较小。少数患者在治疗初期有恶心、腹胀，能自行消失。对肝、肾功能无明显影响。少部分患者用药3～4个月后出现粒细胞减少，故应定期查血象。极个别病例出现耐药性及持续性腹泻时，应停止用药。

**药物相互作用**: 若与阿司匹林及其他水杨酸类制剂、吡嗪酰胺同服，可减弱本品的作用。

**说明**: 国内报道，应用国产苯溴马隆，对血尿酸超过0.357mmol/L的18例患者进行疗效观察，显效6例，治疗后血尿酸降低54％；有效10例，治疗后血尿酸降低38％；无效2例；总有效率89％。服药前后观察肌酐清除率变化表明，本品在血尿酸降低的情况下，肾功能有明显改善。

### 药品9: 蟾酥丸


**类型**: 西药

**相关疾病**: 乳痈

**适应症**: 用于疔毒恶疮，痈疽发背，初起红肿坚硬，麻木疼痛，乳痈肿痛，蝎蜇虫咬伤，热疼痛等症。

**用法用量**: 用醋研磨涂患处。

### 药品10: 复方氨基酸(18AA-V)


**相关疾病**: 蛋白质-能量营养不良 && 肝昏迷

**适应症**: 用于营养不良性水肿(低蛋白血症)、蛋白质-能量营养不良(营养不良)及外科手术前后。

**用法用量**: 成人常规剂量：静脉滴注：1.低蛋白血症：一次500ml，缓慢滴注。2.营养不良：一次500ml，缓慢滴注。3.外科手术前后：一次1500ml，缓慢滴注，滴速为每分钟34～40滴，病情严重者应根据年龄、体重及症状调整剂量，并减慢滴速。老年人剂量：应根据年龄、体重及症状调整剂量，并减慢滴速。儿童：常规剂量：应根据年龄、体重及症状调整剂量，并减慢滴速。

**禁忌**: 1.肝昏迷或有肝昏迷倾向者。2.严重肾功能不全者。3.氨基酸代谢异常者。4.血氮过多者。

**不良反应**: 1.心血管系统：偶有胸闷、心悸等。2.胃肠道：偶有恶心、呕吐等。3.过敏反应：可出现疹样的过敏反应，症状较少。4.其他：(1)偶有发冷、发热或头痛等一般反应。(2)偶可影响肝及肾功能。(3)长期用药可能引起代谢性酸中毒。

**药物相互作用**: 尚不明确。

### 药品11: 扎莫特罗


**类型**: 西药

**相关疾病**: 主动脉瓣狭窄

**适应症**: 适用于治疗慢性心力衰竭，尤其适用于因气喘、疲劳等症状使活动受到限制的心力衰竭病人。

**用法用量**: 口服：每次200mg，2次/d。

**禁忌**: 1.孕妇及哺乳期妇女禁用。2.主动脉瓣狭窄、心肌梗死后患者慎用。

**不良反应**: 本品副作用较少。偶见有胃肠道不适、头痛、眩晕、心悸、心绞痛、肌肉痉挛和急躁等，减量或停药即可消失。

**说明**: 有研究报道，对433例轻至中度充血性心力衰竭或者进行的多中心临床研究中，本品作用显著优于地高辛；另对269例由于气短和疲乏而非由于心肌缺血导致活动受限的心力衰竭患者，进行随机、双盲对照的临床研究中，服用本品患者比服用地高辛患者运动能力增强更明显。本品的双向作用在其他药物中是不多见的，因其特殊的休息时正性肌力作用，使患者运动量由轻度增强到中度时，有降低心肌耗氧量的作用，特别适用于治疗伴有交感神经功能低下的心衰患者。

### 药品12: 氯雷他定/伪麻黄碱


**类型**: 西药

**相关疾病**: 感冒 && 过敏性鼻炎 && 甲状腺功能亢进 && 皮肤瘙痒症

**适应症**: 适用于缓解过敏性鼻炎和感冒的伴发症状，如鼻充血、打喷嚏、流涕、皮肤瘙痒症、流泪等。

**用法用量**: 口服给药：1.成人：每次1片，1天2次。2.儿童： 12岁以上儿童用法与用量参见成人。

**禁忌**: 1.对肾上腺素能制剂或与本药化学结构相似的药物有过敏或特异性反应的患者。2.正在接受单胺氧化酶抑制药治疗或者停止单胺氧化酶抑制药治疗在10天之内的患者。3.有闭角型青光眼、尿潴留、严重高血压、严重冠状动脉疾患或甲状腺功能亢进患者。4.哺乳期妇女。5.12岁以下儿童。

**不良反应**: 1.较常见不良反应有失眠症、口干、头痛和嗜睡。2.偶见紧张、头晕、乏力、精神症状、发音困难、运动功能亢进、性欲减低、感觉异常、震颤、眩晕、味觉异常、激动、淡漠、抑郁、欣快、噩梦、恶心、呕吐、腹部不适、畏食、渴感、排便习惯改变、消化不良、嗳气、痔、一过性肝功能异常、心动过速、潮红、直立性低血压或高血压、心悸、多汗、咽和鼻部刺激症、痤疮、瘙痒、皮疹、荨麻疹、关节痛、支气管痉挛、尿痛、排尿困难、多尿症、尿潴留、虚弱、背痛、小腿痛性痉挛和寒战。

**药物相互作用**: 1.与洋地黄同用，可能增加异位起搏点的活性。2.抗酸药可增加伪麻黄碱的吸收率。3.接受单胺氧化酶(MAO)抑制药的患者使用本药，可能发生高血压反应，甚至引发高血压危象。4.本药含有的伪麻黄碱为拟交感神经药，可降低甲基多巴、美卡拉明(美加明)、利血平和藜芦生物碱的降血压作用。

**说明**: 本品中氯雷他定有明显的抗过敏作用，通过阻断组胺作用减少皮肤瘙痒。由于本品中麻黄碱是一种缩血管药物，对减少鼻黏膜方面有明显作用，故本品作为抗过敏性鼻炎药物使用较广泛。

### 药品13: 法扎溴铵


**适应症**: 适用于外科手术时麻醉辅助用药，如短小手术及诊断处置可能轻度损害者。剖宫产手术可用。

**用法用量**: 静注，初始剂量为0.75～1mg/kg，继续给药为初始剂量的1/4，可延长作用时间1～20min。

**不良反应**: 使用本品0.5mg/kg可发生心动过速，其作用较神经肌肉阻断更长，注射部位可发生局部刺激，在儿童中尤为明显，支气管痉挛和荨麻疹偶有发生。

### 药品14: 硫喷妥钠


**相关疾病**: 支气管哮喘

**适应症**: 临床上广泛应用本品进行基础麻醉、全麻诱导，也用以有效地制止各种原因所致的惊厥状态。

**用法用量**: 临用前用灭菌注射用水配成2.5%溶液。诱导麻醉：4～8mg/kg静注。小儿基础麻醉：15～20mg/kg肌注；3个月以下婴幼儿原则上不用。抗惊厥：2～4mg/kg静注。极量：静注成人一次不超过1g。

**禁忌**: 对呼吸障碍、哮喘、严重休克、萎缩性肌强直、支气管哮喘、血紫质症、营养不良、巴比妥类过敏、新生儿等均禁用。

**不良反应**: 1.易引起呼吸抑制，静注速度应缓慢，而且必须准备好处理呼吸衰竭的应急措施。2.事先给予阿托品对咳嗽、喉和支气管痉挛等不良反应有一定的预防作用。3.剂量过大或注射速度过快易致严重低血压，心血管病人、高血压病人和年老体弱者应用尤应注意。切忌剂量过大、注射速度过快，且须在病人平卧或侧卧时才能给药。4.静脉注射5%的浓溶液可发生血栓性静脉炎及过敏反应。本品溢出脉管可导致组织坏死，注入动脉可产生灼痛，并使前臂和手较长时间退色和指端坏疽。如误入动脉，应立即由动脉注入普鲁卡因和酚妥拉明注射液，以保持该段动脉处于扩张状态，使血流畅通。

**药物相互作用**: 本品与琥珀胆碱混合就迅速水解、沉淀而失效。与氯胺酮、泮库溴铵、哌替啶、麻黄碱、普鲁卡因、苯海拉明以及噻嗪类等相混可产生沉淀。与吗啡合用可发生呼吸抑制，术前用降压药者选用本品要防止低血压发生。与其他中枢神经抑制药合用须减量，同时注意低血压和苏醒延迟。与氯胺酮合用，常见低血压，呼吸慢而浅。与氟烷合用也能引起血压下降。

### 药品15: 磁麻苷


**类型**: 西药

**相关疾病**: 神经衰弱 && 失眠 && 头痛 && 眩晕

**适应症**: 本药适用于肝阳上亢，头痛眩晕，心悸失眠，水肿尿少，以及高血压，神经衰弱，肾炎水肿。

**用法用量**: 1.茶剂：开水冲泡代茶饮，一次3～6g，一日2～3次。2.片剂：口服，一次4片，一日3次。3.颗粒(冲剂)：口服，一次12g，一日2～3次，开水冲服，或遵医嘱，亦可当茶饮。

**说明**: 1.本药药性寒凉，脾虚慢惊者慎用；不宜过量或长期服用，以免中毒。出现心脏毒性反应时，按照洋地黄中毒处理。中毒原因主要是剂量过大，其次是配伍应用不合理。预防中毒要掌握罗布麻根的用量；凡在1～3周内用过洋地黄制剂者，不宜应用；同时注意心电图监测，及时处理。2.应用警示：本品药性缓和，病重者应配合其他药物治疗。用量不宜过大，以免对心脏产生不良作用。 3.简要提示：(1)临床观察表明，本品具有抗衰老、抗血小板聚集、提高免疫球蛋白等作用。(2)临床用于高血压、感冒、慢性气管炎、高脂血症等。

### 药品16: 磺胺二甲嘧啶干混悬剂


**类型**: 西药

**适应症**: 主要用于敏感菌引起的轻症感染，如急性单纯性下尿路感染、急性中耳炎和皮肤软组织感染。

### 药品59: 格列波脲


**相关疾病**: 昏迷 && 糖尿病

**适应症**: 用于40岁以上成年型糖尿病，非胰岛素依赖型而单靠饮食疗法又不能很好控制的糖尿病。

**用法用量**: 口服：初始剂量每次12.5mg，每天1次，以后渐增至常用剂量每天12.5～25mg，最大剂量每天50～100mg，大于25mg时每天须分2次服。长期服药视病情需要而定。

**禁忌**: 严重肾功能不全、对磺脲类不能耐受者禁用。有糖尿病昏迷史及昏迷前的青年型糖尿病者、有酮症酸中毒和孕妇等禁用。

**药物相互作用**: 与双胍类药物合用效果更好。

**说明**: 国内报道，用本品治疗4315例成年2型糖尿病患者，其中2026例属初治病例，能使空腹血糖低于150mg/dl，餐后2h血糖低于150mg/dl者占95%。用本品治疗2型糖尿病37例，连续3～6个月，对治疗前后的有关参数进行比较表明，治疗后糖代谢改善，血浆胰岛素和C肽水平升高(P＜0.01)；高血糖素下降(P＜0.01)；糖化血红蛋白、血液黏度、血小板聚集率降低；血浆支链氨基酸浓度减少，甲皱微循环改善。结果提示本品可增强胰岛β细胞功能，可能有预防糖尿病血管并发症的作用。对格列本脲继发失效者也有效。本品作用强，有较好降糖作用，但用量大，易出现低血糖等不良反应。不作为轻型及老年糖尿病患者首选药物。

### 药品60: 氟甲松龙


**类型**: 西药

**相关疾病**: 单纯疱疹病毒性角膜炎 && 牛痘 && 水痘

**适应症**: 皮质类固醇激素。用于治疗对糖皮质激素敏感的睑球结膜、角膜及其他眼前段组织的炎症。

**用法用量**: 滴入结膜囊内，每次1～2滴，每天2～4次。治疗开始的24～48h可酌情增加至每小时2滴。注意逐步减量停药。点药时用手指压迫泪囊1～2min，避免全身吸收过多。将本品室温密闭保存。

**禁忌**: 1.禁用于急性浅表性单纯疱疹病毒性角膜炎、眼内组织真菌感染、牛痘及水痘感染、及大多数其他病毒性角膜和结膜感染、眼结核以及对该成分过敏者。2.炎眼部长期使用糖皮质激素可能导致真菌感染，角膜溃疡者慎用。

**不良反应**: 长期使用可能引起眼压升高，甚至青光眼，少见视神经损害，后囊膜下白内障，继发性眼部感染，眼球穿孔和延缓伤口愈合。

**药物相互作用**: 尚不明确。
