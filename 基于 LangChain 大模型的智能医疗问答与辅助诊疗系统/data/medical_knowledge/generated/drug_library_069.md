# 药品参考知识库

### 药品1: 阿仑膦酸钠


**相关疾病**: 高钙血症 && 骨质疏松

**适应症**: 用于骨质疏松，恶性肿瘤并发的高钙血症，溶骨性癌转移引起的骨痛及佩古特病。

**用法用量**: 口服：用于绝经期骨质疏松和预防骨折，每次10mg，每天1次。用于预防骨质疏松，每次5mg，每天1次。用于变形性骨炎，每次40mg，每天1次，早餐前30min服用。

**禁忌**: 1.高钙血症患者禁用。2.对二磷酸盐类药物过敏者禁用。

**不良反应**: 1.常见有胃刺激症状，如恶心、呕吐、腹痛、胸骨后疼痛、消化不良、咽痛、吞咽疼痛、胸闷、头晕及轻微肝肾功能改变。2.少见的不良反应有低血钙现象及听力丧失。长期应用未见在体内蓄积。

**药物相互作用**: 不得与其他种类二膦酸盐类药物合并使用。

**说明**: 对骨矿物质密度的影响在四项为期2年或3年疗程的双盲、安慰剂对照的临床研究中，证明了每天给予10mg本品对绝经后骨质疏松症妇女的疗效。因此，不论患者以前是否出现过脊椎骨折，本药均能降低其骨折发生率。在骨折干预试验研究中，对髋部和腕部骨折的累积发生率的3年研究结果显示，本品治疗患者的这些类型的骨折发生率在各时间点均低于安慰剂治疗的患者。本品将髋部骨折妇女的比例降低了51%，而腕部骨折降低了48%。在五项合并的骨质疏松症治疗研究中观察到的髋部和腕部骨折降低比例相近。总的来说，这些结果表明，本药能有效的降低包括脊椎和髋部在内的骨质疏松性骨折发病率最高的部位的骨折发生率。270名用本品剂量每天1～20mg，治疗一、二或三年的绝经后骨质疏松症患者的骨组织学检查表明，与安慰剂比较，治疗组出现了正常的骨钙化和结构，以及所希望的骨转换率降低。连同在长期给予阿仑膦酸钠的大鼠和狒狒所观察到的正常骨组织学和骨强度增加的结果，这些数据表明，本药治疗期间形成的骨骼是正常的。

### 药品2: 双歧杆菌/嗜酸性乳杆菌/粪链球菌


**类型**: 西药

**相关疾病**: 便秘 && 腹泻

**适应症**: 主要用于肠道菌群失调引起的腹泻、慢性腹泻，抗生素治疗无效的腹泻及便秘。

**用法用量**: 口服：成人，每日2～3次，每次4片；6个月内婴儿，1次1片；6个月至3岁者，1次2片；3～12岁者，1次3片，每日2～3次。用温开水或温牛奶送服。

**禁忌**: 对本品过敏者禁用。

**药物相互作用**: 1.与制酸药、抗生素合用，应间隔2h。2.不宜与铋剂、鞣酸、活性炭、酊剂等合用。

**说明**: 本品为青春型双歧杆菌经驯化培育而成的生产菌株，经发酵，精制而得。可补充对人体有益的正常的肠蠕动，减少内毒素来源，降低血内毒素水平；还可产生多种生物酶，使蛋白质转变成氨基酸，脂肪转变成为脂肪酸，糖特别是乳糖分解成为乳酸，从而促进这三大营养素的吸收与利用。对肝炎病人，能够改善肝功能，促进肝细胞功能的恢复，对肝硬化病人，能够改善肝脏蛋白质的代谢，减轻肝脏负担，发挥保肝、护肝等作用。本品能在体内合成多种维生素，如维生素B族和烟酸，能补充多种微量元素和氨基酸，还有利于分解、转化和利用蛋白质及某些无机盐的吸收与利用。本品对T、B淋巴细胞，骨髓多能干细胞等有一定的激活作用。

### 药品3: 奥拉西坦胶囊


**类型**: 西药

**适应症**: 适用于轻中度血管性痴呆、老年性痴呆以及脑外伤等症引起的记忆与智能障碍。

**用法用量**: 口服，每次2粒(800mg)，每日2～3次，或遵医嘱。

**禁忌**: 对本品过敏者禁用。

**不良反应**: 据国外文献报道，奥拉西坦的不良反应少见，少数患者出现精神兴奋和睡眠异常。本品在357例Ⅱ期临床研究中，仅个别患者出现恶心和胃部不适。

**药物相互作用**: 尚不明确。

### 药品4: 孕二烯酮


**类型**: 西药

**相关疾病**: 高脂血症 && 黄疸 && 静脉血栓 && 乳腺癌 && 糖尿病

**适应症**: 临床上本品与炔雌醇组成复合片或三相片用作短效口服避孕药，用于女性避孕。

**用法用量**: 单相短效口服避孕药应于月经第5天开始服用，每天服1片，连服22天，停药后3～7天内行经；于行经的第5天再服下一个周期的药。产后或流产后在月经来潮后再服。三相片，每6天、5天和10天依次服用不同含量的药片，每天1片。

**禁忌**: 禁用于乳腺癌、生殖器官癌、肝功能异常或近期有肝病或黄疸史、肾病、深部静脉血栓病、脑血管意外、高血压、心血管病、糖尿病、高脂血症、精神抑郁症及40岁以上妇女、孕妇、乳妇。

**不良反应**: 1.生殖系统类早孕反应：表现为恶心、呕吐、困倦、头晕、食欲减退；突破性出血(多发生于漏服时)；闭经(少数停药后仍继续闭经)。2.内分泌系统：体重增加；面部色素沉着、乳房胀。3.消化系统：肝功能损害；肝良性腺瘤相对危险性增高。4.心血管系统：高血压；年龄大于35岁的吸烟妇女患缺血性心脏疾患危险性增加。5.高剂量雌激素复方片有增加血栓栓塞病的危险性，有证据显示含孕二烯酮的口服合成避孕药同少见的患静脉血栓栓塞有关。6.其他：(1)精神压抑、头痛、疲乏。(2)据报道孕二烯酮有很轻微的雄激素作用。7.一般在睡前服，不良反应可减轻。

**药物相互作用**: 1.影响避孕效果的药物：(1)抗生素类抑制肠内细菌繁殖，减少激素结合物的分解，减少肠肝循环。(2)诱导肝药酶的药物如利福平、催眠和抗惊厥药、解热镇痛药。(3)三环类抗抑郁药在肝脏与本品竞争共同的代谢酶。(4)抗癫痫药：合用苯丙胺酯可增强低剂量的复合口服避孕药中孕二烯酮的清除率，从而可能降低避孕药的效果。2.与避孕药合并应用，药效受影响的药物：(1)降压药的降压效果降低。(2)抗凝药的抗凝效果降低。(3)降糖药(如胰岛素及口服降糖药)控制糖尿病的疗效降低。(4)三环类抗抑郁药的效果增强。

**说明**: 本品为国外研制成的新一代避孕药中较有前途的一种，是迄今孕激素作用最强而使用剂量最低的一种避孕药。其孕激素活性为左炔诺孕酮的2倍，并且无雄激素和雌激素活性，有抗雌激素作用。本品也是迄今惟一发现有抗醛固酮活性的孕激素。能抑制促性腺激素分泌，抑制排卵作用强度与炔诺孕酮相似。由于其避孕效果可靠、周期控制好，对脂代谢能产生有利影响，提高高密度脂蛋白(HDL)，故为目前最为理想的一种口服避孕药。

### 药品5: 替加氟注射液


**类型**: 西药

**适应症**: 主要用于治疗消化道肿瘤，如胃癌、直肠癌、胰腺癌、肝癌，亦可用于乳腺癌。

**用法用量**: 单药成人一日剂量800～1000mg或按体重一次15～20mg/kg，溶于5%葡萄糖注射液或0.9%氯化钠注射液500ml中，一日1次静滴，总量20～40g为一疗程。也可与其他抗肿瘤药物联合应用。

**禁忌**: 禁用。

**不良反应**: 骨髓抑制反应轻，有白细胞、血小板下降。神经毒性反应有头痛、眩晕、共济失调、精神状态改变等。少数病人恶心、呕吐、腹泻、肝肾功能改变。局部注射部位有静脉炎、肿胀和疼痛。偶见发热、皮肤瘙痒、色素沉着。

**药物相互作用**: 替加氟呈碱性且含碳酸盐，避免与含钙、镁离子及酸性较强的药物合用。

### 药品6: 比马前列胺


**类型**: 西药

**适应症**: 本品用于开角型青光眼、使用其他降眼压药物无效或不能耐受的眼压升高患者。

**用法用量**: 滴眼，每晚1滴，频繁使用可降低本品的降眼压作用。首次用药约4h后眼压开始降低，最大效果出现在8～12h内。

**禁忌**: 对本品过敏者禁用。

**不良反应**: 本品能通过增加黑色素细胞中的黑色素小体(色素颗粒)，逐渐改变睫毛的长度和厚度、色素沉着和睫毛数量，使睫毛生长、虹膜和眶周组织的色素沉着增加。临床试验表明，15%～45%的患者最常见的不良反应为结膜充血、睫毛生长和眼睛瘙痒。约3%患者因结膜充血而中止治疗。其他不良反应还有眼干、视觉障碍、眼灼伤、异物感、眼痛、眶周皮肤色素沉着、睑炎、白内障、浅表点状角膜炎、眼睑红斑、眼刺激感、睫毛变黑。不到1%的患者会发生虹膜炎。约低于10%的患者出现系统性不良反应(起初为感冒和上呼吸道感染)，约1%～5%的患者出现头疼、肝功能异常、虚弱和多毛。

### 药品7: 注射用棓丙酯


**类型**: 西药

**适应症**: 用于预防和治疗脑血栓、冠心病以及外科手术后的并发症—血栓性深静脉炎等。

**用法用量**: 静脉滴注：一日1次，每次120～180mg(2～3支)。使用时在每瓶中加入生理盐水5ml，振摇使完全溶解后，加至250～500ml生理盐水或5%葡萄糖注射液中混匀，缓缓静脉滴注；10～15天为一疗程。当室温低于15℃时，每瓶可用2ml丙二醇注射液，振摇使完全溶解后使用。

**禁忌**: 对本品过敏者禁用。

**不良反应**: 少数患者静滴后有一过性心率减慢或谷丙氨基转移酶轻度增高，在停药1～2周内可自行恢复正常。

**药物相互作用**: 未见报道。

### 药品8: 牛黄解毒片


**类型**: 中药

**适应症**: 适用于上焦实热引起的咽喉肿痛。牙龈肿痛，口舌生疮，目赤肿痛，大便不通。

**用法用量**: 口服：每次2片或1丸，每天2～3次，温开水送服。

### 药品9: 环己西林


**适应症**: 临床适用范围与氨苄西林相同，适用于敏感菌所致呼吸系统、泌尿系统感染等。

**用法用量**: 口服给药：成人，每天1～2g，分4次给予；儿童，每天50～100mg/kg，分4次给予。

**禁忌**: 对本品及青霉素过敏者禁用。

**不良反应**: 与氨苄西林相同，应用前须做本品或青霉素过敏试验，方法同青霉素。

**药物相互作用**: 参见氨苄西林，但丙磺舒对本品排出的抑制作用不明显。

**说明**: 临床作用范围与氨苄西林相同。临床有效率及细菌清除率与氨苄西近似。

### 药品10: 盐酸依普拉酮片


**类型**: 西药

**适应症**: 适用于急、慢性支气管炎、哮喘、肺炎、肺结核、肺气肿等疾病的镇咳和祛痰。

**用法用量**: 口服。一次40～80mg（1～2片），一日3～4次。儿童剂量减半。

**不良反应**: 少数患者可有头昏、口干、胃部不适及恶心等。

**药物相互作用**: 尚不明确

### 药品11: 菊糖


**适应症**: 采用菊糖清除试验测量肾小球滤过率(GFR)，也可用于测定细胞外液容量。

**用法用量**: 静脉输注5～7.5g，加入0.9%氯化钠注射液500ml中，以恒速输注。

**不良反应**: 本品一般无毒副作用。制剂不纯，有时可引起热原反应。

### 药品12: 萘甲唑林


**类型**: 西药

**相关疾病**: 变态反应性鼻炎 && 变应性鼻炎 && 急性鼻炎 && 慢性鼻炎

**适应症**: 用于急性鼻炎、慢性鼻炎，鼻窦炎，也可用于变应性鼻炎（变态反应性鼻炎）。

**用法用量**: 滴鼻：2滴/次，6小时1次。

**不良反应**: 长期应用可出现反弹性充血，也可出现萎缩性鼻炎。且可有局部刺激感，恶心、头痛。滴药过多及吞咽过多可致中毒，出现心率减慢、出汗、嗜睡、体温下降、昏迷等。高血压及甲亢者慎用。

### 药品13: 金芪降糖片


**类型**: 中药

**相关疾病**: 非胰岛素依赖型糖尿病

**适应症**: 用于轻、中型非胰岛素依赖型糖尿病，症见口渴喜饮，易饥多食，气短乏力等。

**用法用量**: 饭前0.5h口服。每次7～10片，每天3次，2个月为1个疗程。

### 药品14: 阿托伐醌/氯胍


**相关疾病**: 恶性疟疾 && 脑型疟疾

**适应症**: 预防和治疗脑型疟疾(恶性疟疾) (包括对氯喹已产生耐药性的脑型疟疾)。

**用法用量**: 1.预防恶性疟疾：在进入疫区前1～2天开始服用本品，且在离开疫区后继续用药7天。成人剂量为阿托伐醌250mg，氯胍100mg，1次/d。体重11～20kg儿童给予125mg和50mg，均为1次/d，体重＞40kg儿童用量同成人。2.治疗恶性疟疾：成人剂量为阿托伐醌1g，氯胍40mg，1次/d，连用3天；或500mg和200mg，每12小时1次，连用3天。体重11～20kg的儿童给予250mg和100mg；体重21～30kg儿童给予500mg和200mg；体重31～40kg儿童给予750mg和300mg，均为1次/d，连用3天。＞40kg儿童用量同成人。

**禁忌**: 1.对本品过敏者、哺乳者禁用。2.肝功能不全者和肾功能明显不全者慎用。

**不良反应**: 1.可能发生过敏反应、瘙痒、ALT和ASH、升高(大多数在28天内恢复正常)。2.成人可出现无力、厌食、恶心、呕吐、腹痛和腹泻，儿童的呕吐发生率更高。

**药物相互作用**: 1.P450 CY2C19的酶诱导剂或酶抑制剂都可能影响本品的血药浓度。2.甲氧氯普胺、利福平或四环素可降低本品血药浓度(指阿托伐醌)。

### 药品15: 阿瑞吡坦


**适应症**: 用于化疗后的急性和延迟性恶心或呕吐发作。还可用于重度抑郁症(伴焦虑)。

**用法用量**: 口服给药。1．化疗诱发的恶心和呕吐：初始剂量为首日125mg，化疗前1h服用；第2～3日，每日80mg，化疗前1h服用。2．用于重度抑郁症(伴焦虑)每次300mg，每日1次。肾功能不全、轻到中度肝功能不全的患者、老年人、晚期肾病(ERSD)进行血液透析的患者无需调整剂量。
