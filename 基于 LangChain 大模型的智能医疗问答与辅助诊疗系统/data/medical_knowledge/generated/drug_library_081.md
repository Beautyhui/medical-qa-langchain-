# 药品参考知识库

### 药品1: 注射用利巴韦林


**类型**: 西药

**适应症**: 抗病毒药。用于呼吸道合胞病毒引起的病毒性肺炎与支气管炎。

**用法用量**: 用氯化钠注射液或5%葡萄糖注射液稀释成每1ml含1mg的溶液后静脉缓慢滴注。成人一次0.5g，一日2次，小儿按体重一日10～15mg/kg，分2次给药。每次滴注20分钟以上，疗程3～7日。

**禁忌**: 对本品过敏者、孕妇禁用。

**药物相互作用**: 本品与齐多夫定同用时有拮抗作用，因本品可抑制齐多夫定转变成活性型的磷酸齐多夫定。

### 药品2: 艾司唑仑片


**类型**: 西药

**适应症**: 主要用于抗焦虑、失眠。也用于紧张、恐惧及抗癫痫和抗惊厥。

**用法用量**: 成人常用量 镇静，一次1～2mg（1～2片），一日3次。催眠，1～2mg（1～2片），睡前服。抗癫痫、抗惊厥，一次2～4mg（2～4片），一日3次。

**禁忌**: 慎用者：1．中枢神经系统处于抑制状态的急性酒精中毒。2．肝肾功能损害。3．重症肌无力 。4．急性或易于发生的闭角型青光眼发作。5．严重慢性阻塞性肺部病变。

**不良反应**: 常见的不良反应：1.首次服用本品初期可能出现过敏性休克（严重过敏反应）和血管性水肿（严重面部浮肿）。服用本品可能引起睡眠综合征行为，包括驾车梦游、梦游做饭和吃东西等潜在危险行为。2.口干、嗜睡、头昏、乏力等，大剂量可有共济失调、震颤。3.罕见的有皮疹、白细胞减少。4.个别病人发生兴奋，多语，睡眠障碍，甚至幻觉。停药后，上述症状很快消失。5.有依赖性，但较轻，长期应用后，停药可能发生撤药症状，表现为激动或忧郁。

**药物相互作用**: 1．与中枢抑制药合用可增加呼吸抑制作用。2．与易成瘾和其他可能成瘾药合用时，成瘾的危险性增加。3．与酒及全麻药、可乐定、镇痛药、吩噻嗪类、单胺氧化酶A型抑制药和三环类抗抑郁药合用时，可彼此增效，应调整用量。4．与抗高血压药和利尿降压药合用，可使降压作用增强。5．与西咪替丁、普奈洛尔合用本药清除减慢，血浆半衰期延长。6．与扑米酮合用由于减慢后者代谢，需调整扑米酮的用量。7．与左旋多巴合用时，可降低后者的疗效。8．与利福平合用，增加本品的消除，血药浓度降低。9．异烟肼抑制本品的消除，致血药浓度增高。10．与地高辛合用，可增加地高辛血药浓度而致中毒。

**类型**: 西药

**相关疾病**: 类风湿性关节炎 && 荨麻疹 && 增生性骨关节病 && 支气管哮喘

### 药品12: 茶碱


**类型**: 西药

**相关疾病**: 急性心肌梗死 && 心源性哮喘 && 支气管哮喘

**适应症**: 主要用于支气管哮喘和喘息样支气管炎。也可用于心源性哮喘。

**用法用量**: 口服：每12小时一次，每次0.2～0.3g，或每24～36小时1次，每次0.4g。

**禁忌**: 茶碱过敏者、消化道溃疡、急性心肌梗死、休克患者禁用。

**不良反应**: 大量的临床观察证明：血清茶碱浓度低于15mg/L者，基本上没有毒性反应；15～25mg/L者，偶有恶心、呕吐、失眠等反应；25mg/L以上者，毒性反应发生率为75%；血药浓度在40mg/L以上，可能发生心动过速、心律失常、抽搐和呕吐咖啡样物（呕血，多见于幼儿）等中毒反应；严重者可发生呼吸心跳停止而骤然死亡；但这种病例多见于快速静脉推注茶碱病例，慢速静脉点滴者少见。

**药物相互作用**: 1.地尔硫卓、维拉帕米可干扰茶碱在肝内的代谢，可增加茶碱的血药浓度和毒性。2.西咪替丁、雷尼替丁可降低茶碱的肝脏清除率，合用时可增加茶碱的血药浓度和（或）毒性。3.某些抗菌药物，如大环内酯类的红霉素，喹诺酮类的依诺沙星、环丙沙星、氧氟沙星，克林霉素，林可霉素等可降低茶碱清除率，增高其血药浓度，尤以依诺沙星为著。当茶碱与上述药物配伍用时，应适当减量。4.美西律可减低茶碱清除率，增加血浆中茶碱浓度，合用时需调整茶碱的剂量。5.咖啡因及其他黄嘌呤类药可增加茶碱的作用和毒性。6.氟康唑、他克林、噻苯达唑、噻氯匹定、维洛沙嗪、双硫仑、羟乙桂胺等可使茶碱血药浓度升高，出现过量危险。7.茶碱与沙丁胺醇合用有协同作用，同时也增强不良反应。8.别嘌醇可降低茶碱的清除率，增加本药的毒性。9.苯巴比妥、苯妥英钠及卡马西平，可刺激茶碱的肝脏代谢，加快茶碱清除率；同时，茶碱也干扰苯妥英钠的吸收，导致两者血浆中浓度均下降，合用时应调整剂量。10.卡介苗、流感病毒疫苗可降低茶碱的清除率，增加茶碱的毒性。11.阿糖腺苷可升高茶碱血药浓度。12.抗甲状腺药可减慢机体对茶碱的代谢，从而使茶碱血药浓度升高，作用增强。13.己酮可可碱可使茶碱血药浓度升高，出现过量危险。14.干扰素可降低茶碱清除率。15.茶碱能增强呋噻米的利尿作用。16.茶碱与利血平合用，可使心率加快。17.稀盐酸可减少茶碱的吸收。18.硫糖铝可减少茶碱的吸收。19.茶碱与磺吡酮合用，茶碱的血药浓度降低。20.氨鲁米特可增加茶碱的清除率。21.药用炭可降低茶碱的血药浓度。22.甲状腺素可加速茶碱的代谢，从而使其血药浓度降低，作用减弱。23.茶碱与锂盐合用，可使锂盐的肾排泄增加，影响锂盐的作用。吸烟会影响茶碱的作用。24.使用茶碱时，应避免饮用含大量咖啡因的饮料，避免大量食用巧克力，这些食物均会增加茶碱的不良反应。

**说明**: 国内报道，用茶碱控释片治疗哮喘性支气管炎和慢性支气管炎210例，总有效率为96%。用国产茶碱缓释片治疗60例支气管哮喘，口服200mg，每天2次，共7天。有效率75%，显效为28%。主要不良反应：胃部不适、疼痛、恶心、呕吐(15%)、心悸(8%)、轻度头痛(6%)、局部皮疹(5%)、血糖略有升高(3%)。以上不良反应经停药或疗程结束后均恢复正常。

### 药品13: 阿法双酮


**适应症**: 适用于全麻诱导、短小手术、外科处置，也可用作全麻辅助药。

**用法用量**: 静注：首剂量为0.05～0.1ml/kg，于1min左右静注完毕，维持量为首剂量的1/2。

**禁忌**: 严重肝功能不全者及孕妇禁用。

**不良反应**: 过敏反应发生率很高，静注后可出现一过性血压下降，对呼吸和循环系统的抑制作用较小。

**药物相互作用**: 本药与肌松药无协同作用。对肾上腺素引起的心律失常有拮抗作用。

**说明**: 阿法双酮为静脉麻醉药。麻醉迅速，静注后半分钟入睡，5～10min意识消失，适用于全麻诱导、短小手术、外科处置，也可用作全麻辅助药。严重肝功能不全者及孕妇禁用。不适用于有强烈疼痛手术的麻醉。过敏反应发生率很高，临床使用要引起重视。
